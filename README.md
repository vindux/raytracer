# CPU Raytracer written in Java

### Preview

    [![](https://raw.githubusercontent.com/javacpu/cpu-raytracer/master/docs/preview.png)](

### Features

- Shapes
    - Plane, Sphere, Square
- Shaders
  - Lambert
  - Phong
- Lights 
  - Area
  - Point
- Reflection, Refraction
- Threading (multicore rendering)
- Anti Aliasing (uses supersampling)
